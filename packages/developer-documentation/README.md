# @continuum-dev/developer-documentation

Bundled developer documentation for consumers such as the marketing site: the repository root `README.md`, the five public guides under `docs/` (`AI_INTEGRATION.md`, `HOW_CONTINUITY_DECISIONS_WORK.md`, `QUICK_START.md`, `INTEGRATION_GUIDE.md`, `VIEW_CONTRACT.md`), and an explicit allowlist of package READMEs plus `packages/session/STREAMING.md` (see `scripts/generate-corpus.mjs`). Other markdown is not embedded. Generated at build time. Install from npm rather than importing markdown from the SDK monorepo.
