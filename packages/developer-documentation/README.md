# @continuum-dev/developer-documentation

Bundled developer documentation for consumers such as the marketing site: the repository root `README.md`, the four public guides under `docs/` (`QUICK_START.md`, `INTEGRATION_GUIDE.md`, `AI_INTEGRATION.md`, `VIEW_CONTRACT.md`), and an **explicit allowlist** of package READMEs plus `packages/session/STREAMING.md` (see `scripts/generate-corpus.mjs`). Other markdown is not embedded. Generated at build time. Install from npm rather than importing markdown from the SDK monorepo.
