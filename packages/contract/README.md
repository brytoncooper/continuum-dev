# @continuum/contract

This package defines the core data structures and interfaces for the Headless Continuity Runtime.

It serves as the "Protocol" between the Host Application (UI) and the Continuity Runtime (Logic).

## Key Concepts

### ContinuitySnapshot
The primary data structure that captures the full state of the interface at a specific moment in time.

```typescript
interface ContinuitySnapshot {
  schema: SchemaSnapshot; // The "Shape of the World" (From AI)
  state: StateSnapshot;   // The "User's Intent" (From User)
}
```

### SchemaSnapshot
Represents the structural definition of the interface, typically generated by an AI agent.

### StateSnapshot
Represents the user's accumulated input and interaction state applied against the schema.
